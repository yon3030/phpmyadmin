Reduce motion query

Metadata

When to Use

Use on web pages that include CSS animation motion.

Description

The objective of this technique is to demonstrate a programmatic method of disabling animation motion through the prefers-reduced-motion CSS Media Query.

Examples

'prefers-reduced-motion' CSS Media Query

Users can indicate their motion preference for interfaces in their system and the 'prefers-reduced-motion' CSS Media Query respects that choice.

@media (prefers-reduced-motion: reduce) { }

Working example of 'prefers-reduced-motion' CSS Media Query

Tests

Procedure

For each page element that applies motion through CSS:

  1. Check whether the motion is disabled in the prefers-reduced-motion CSS Media Query;
  2. Enable the 'Reduce Motion' setting in your system.

Expected Results

  • Motion should stop on the applied page elements.
