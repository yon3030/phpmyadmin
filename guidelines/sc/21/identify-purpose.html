Contextual Information

AAA

New

This Success Criterion is unchanged from the previous Working Draft, and discussion about comments received is ongoing. The Working Group is interested in additional comments on this version of the Success Criterion. If you have strong views about the output of this discussion, please follow discussion in progress on Contextual Information issues.

In content implemented using markup languages, contextual information for controls, symbols, and regions can be programmatically determined using a publicly available vocabulary.
