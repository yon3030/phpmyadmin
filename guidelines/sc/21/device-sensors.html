A

New

This is a proposed Success Criterion that has not yet been formally approved by the Working Group. Discussion of the issue is available in Issue 67. To file comments on this proposal, please raise new issues for each discrete comment in GitHub.

All functionality of the content can be operated without requiring specific device sensor information, unless the device sensor is essential for the function and not using it would invalidate the activity.

The Working Group has identified questions of scope for this Success Criterion, and is exploring rewording to resolve this. The main question is about the relationship to indirect motion associated with operating a keyboard, pointer, or assistive technology. Draft new wording is available at https://github.com/w3c/wcag21/issues/67#issuecomment-325716721
