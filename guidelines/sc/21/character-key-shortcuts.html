Character Key Shortcuts

A

New

If a keyboard shortcut is implemented in content using only letter (including upper- and lower-case letters), punctuation, number, or symbol characters, then at least one of the following is true unless the keyboard shortcut for a user interface component is only active when that component has focus:

Turn off
A mechanism is available to turn the shortcut off;
Remap
A mechanism is available to remap the shortcut to use one or more non-printable keyboard characters (e.g. Ctrl, Alt, Shift, etc.) or the Space or Enter keys.
