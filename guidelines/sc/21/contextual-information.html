Contextual Information

AAA

New

This Success Criterion is unchanged from the previous Working Draft. The Working Group received comments but was not able to reach consensus on language to address the comments. The Working Group is interested in additional comments on this version of the Success Criterion.

To file comments on this proposal, please raise new issues for each discrete comment in GitHub.

In content implemented using markup languages, contextual information for controls, symbols, and regions can be programmatically determined using a publicly available vocabulary.
